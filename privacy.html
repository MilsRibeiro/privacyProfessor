Política de Privacidade do Aplicativo Mils Educador Físico professor

Última atualização: 14/07/2026

Esta é a política de privacidade oficial do aplicativo Mils Educador Físico professor, pacote mils.educador.professor. Ela descreve como coletamos, usamos, retém e protegemos os dados dos usuários.

1. Dados coletados

  • Câmera: usada apenas para capturar imagens necessárias para login e autenticação biométrica do professor. Essas imagens não são armazenadas de forma permanente.
  • Dados biométricos: utilizados exclusivamente para autenticação local no dispositivo. Esses dados não são enviados a servidores externos ou compartilhados com terceiros.
  • Dados de conta e login: informações necessárias para autenticar o usuário e permitir o acesso ao app.
  • Dados de uso: informações sobre preferências e configurações do app podem ser armazenadas localmente para melhorar a experiência do usuário.
  • Dados de armazenamento: o app pode gravar dados no dispositivo usando armazenamento local e preferências.

2. Finalidade do uso

  • Permitir acesso seguro ao aplicativo.
  • Autenticar o usuário professor por meio de login e biometria.
  • Manter preferências e configurações do usuário para melhor experiência.
  • Garantir o funcionamento correto do aplicativo, incluindo acesso à câmera e armazenamento quando solicitado.

3. Retenção de dados

Os dados coletados pelo app são mantidos apenas pelo tempo necessário para cumprir as finalidades descritas nesta política. O app não retém dados desnecessários.

Dados armazenados localmente no dispositivo permanecem até que o app seja desinstalado ou até que o usuário solicite exclusão.

Se existirem dados associados à conta em servidores remotos, eles são mantidos enquanto a conta estiver ativa e são excluídos em até 30 dias úteis após solicitação de exclusão do usuário.

4. Tratamento de dados

Os dados do usuário são tratados de forma segura e de acordo com as finalidades deste app. Aplicamos as seguintes práticas de tratamento:

  • Minimização de dados: coletamos apenas os dados necessários para autenticação e funcionamento do app.
  • Processamento local: dados biométricos são utilizados apenas localmente no dispositivo para autenticação.
  • Uso restrito de câmera: as imagens da câmera são processadas somente para login e autenticação e não são armazenadas de forma permanente.
  • Proteção de dados: usamos medidas técnicas para proteger dados sensíveis enquanto são tratados e armazenados.

5. Compartilhamento de dados

  • Nenhum dado é compartilhado com terceiros sem o consentimento explícito do usuário.
  • Dados biométricos permanecem no dispositivo e não são transferidos para outros serviços.

5. Segurança

  • Aplicamos medidas técnicas para proteger dados sensíveis.
  • O app utiliza permissões de câmera e armazenamento estritamente para as funcionalidades descritas nesta política.
  • Dados biométricos não são armazenados em servidores externos sem consentimento explícito.

6. Direitos do usuário

  • Usuários podem solicitar acesso, correção ou exclusão de seus dados a qualquer momento.
  • Dados locais podem ser removidos ao desinstalar o app do dispositivo.
  • Para exclusão de dados vinculados à conta ou servidor, entre em contato pelo email abaixo.

Para solicitar exclusão de dados ou esclarecer dúvidas sobre esta política, envie um email para:

milscodgos@gmail.com

Inclua no email seu nome, o email usado no app, o pacote do app (mils.educador.professor) e, se possível, o identificador do usuário no app. Processaremos a solicitação em até 30 dias úteis.

7. Alterações nesta política

Esta política pode ser atualizada. A data acima será alterada em caso de mudanças relevantes.
